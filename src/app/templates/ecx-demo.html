Event Calendar X

About The App

Event Calendar X allows a Shopify merchant to easily embed an event calendar on any page in their store as well as provides an interface for creating and editing events. Both of these features are fully integrated into the Merchant's store and do not require the user to leave their Shopify account at any point.

Adding and removing events takes place in the 'Admin Panel', which can be found by logging in to your Shopify account and selecting 'Event Calendar X' under 'Apps'.

Adding a calendar to your store takes place through the Shopify Theme Editor where you will find the Event Calendar option in the editing panel.

Check out the demo below to see what the app looks and feels like, and stay tuned for more feature releases over the coming weeks...including hopefully a better name :)
